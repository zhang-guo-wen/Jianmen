# Jianmen — 轻量级堡垒机

**Jianmen**（剑门）是一个 Go 语言编写的轻量级堡垒机（Bastion Host），提供 SSH/SFTP 代理、数据库代理、终端录像、命令审计和 Web 管理界面。

> 当前处于 MVP / 一阶段核心代理状态，尚未发布正式版本。

## 功能特性

### 资源与账号管理

- **主机资源管理** — 统一维护主机及其登录账号，支持分组、状态、有效期、密码与私钥认证。
- **数据库资源管理** — 管理 MySQL、PostgreSQL、Redis 实例及数据库账号，资源变更可动态刷新代理配置。
- **应用与平台账号** — 支持应用代理配置，连接内网网站、平台账号集中管理，以及面向用户或角色的账号共享。
- **账号级资源模型** — 主机和数据库作为资源容器，连接、授权、审计围绕具体账号执行。

### 安全连接

- **SSH Shell 代理** — 支持密码、公钥和 keyboard-interactive 认证，以及 PTY、窗口 Resize、Signal 转发。
- **SFTP 文件代理** — 提供语义层文件代理，兼容 Xftp、WinSCP、FileZilla 等主流客户端。
- **多协议数据库代理** — 支持 MySQL、PostgreSQL、Redis 连接代理，统一执行身份识别、资源授权和会话控制。
- **快速连接** — 集中展示连接地址、堡垒机账号和连接命令，支持 SSH、SFTP、数据库客户端及 Web Terminal。
- **临时连接密码** — 按资源生成限时、一次性连接密码，避免在连接过程中暴露目标资源凭据。
- **本地 SSH 客户端** — 可配置并调用系统默认客户端、Xshell、PuTTY 等本地程序快速发起连接。

### 权限与可见性

- **细粒度 RBAC** — 支持用户、用户组、角色、权限和资源授权，覆盖主机、数据库、账号、应用及资源分组。
- **资源级授权** — 可直接授权主机或数据库容器，也可仅授予具体账号的连接权限。
- **按权限展示** — 管理页面、资源列表和连接入口只展示当前用户有权访问的内容。
- **创建者自动授权** — 用户新增资源后自动获得对应资源的管理权限。

### 审计与追溯

- **终端录像** — 使用 asciinema v2 兼容格式记录 SSH 会话并支持在线回放。
- **命令审计** — 解析交互式 Shell 命令，保留执行时间、会话和识别置信度。
- **文件审计** — 记录 SFTP 文件操作，并按文件句柄统计上传、下载和读写字节。
- **数据库审计** — 记录数据库连接和可观察的查询事件，支持按会话检索。

### 管理与集成

- **Web 管理界面** — 基于 Vue 3 与 Element Plus，提供资源、账号、连接、审计、权限和配置管理。
- **用户配置中心** — 主题、终端字体与本地 SSH 客户端配置按当前用户保存。
- **REST API** — 使用 Bearer Token 认证，可用于自动化脚本、CLI 和第三方系统集成。
- **跨平台构建** — 提供 Windows 与 Linux 构建脚本，可生成包含前端资源的独立二进制程序。

## 后续开发计划
- 数据库在线连接，文本展示查询内容即可
- 在线xftp管理，管理文件的上传、下载、重命名、编辑文件功能
- AI功能，让AI能连接操作服务器，让AI连接操作数据库，做危险命令连接
- 支持window服务器管理，支持远程桌面协议，支持回放审计
### 现有功能优化
- 新增临时授权功能，对一个账号临时授权给其他人，不是新建用户账号而是创建了一个临时session，在权限管理可以看到临时授权，包括sessionID，授权用户，授权有效期，授权的资源名称，授权的账户名称，授权的开始时间，授权的备注。有效期包括1天，3天，7天，最多7天。
- 审计日志，点击跳转到审计日志记录里面，进行搜索过滤
- 在线会话，跳转到审计日志里面，审计日志页面增加一个在线会话的tab页面。看到实例，协议，账号，操作者，开始时间。操作有回放，日志，断开三个按钮。断开会中断当前会话。
- 主机和数据库菜单的更多的的权限管理按钮，点击弹窗，展示能连接这个主机或者数据库的账户和临时授权会话列表，支持分页和搜索。展示字段有，sessionID，用户/授权用户，用户组，有效期，
- 用户默认设置有效期，除管理员，默认是一年。到期则自动停用，需要重新激活。支持设置成长期。
- 更多分组管理，包括应用分组，和主机和数据库一样的属于资源
- 优化ssh在线终端的提示功能

```
│  ┌─ 访问控制层 ──────────────────────┐ │
│  │  用户认证 · MFA · 资产解析 · RBAC │ │
│  ├─ 协议代理层 ──────────────────────┤ │
│  │  SSH Server · SFTP · DB Proxy     │ │
│  ├─ 审计层 ──────────────────────────┤ │
│  │  终端录像 · 命令解析 · 文件审计    │ │
│  ├─ 存储层 ──────────────────────────┤ │
│  │  元数据库 · 录像文件 · 索引        │ │
│  └────────────────────────────────────┘ │
└────────────────────────────────────────┘
        │
        ▼
   目标主机 / 数据库
```
