# 配置示例

- `config.example.json`：独立二进制部署的基础示例。
- `config.docker.json`：容器镜像内置的安全默认配置。
- `config.docker.local.json`：Windows 本机容器运行配置。
- `config.docker.web-rdp.example.json`：启用 Web RDP 的容器配置。
- `config.docker.proxy.example.json`：由反向代理终止 Web TLS 的容器配置。

实际密钥、密码和本地运行配置不要提交到仓库。Windows 本机启动请使用仓库根目录的 `start.ps1`。
